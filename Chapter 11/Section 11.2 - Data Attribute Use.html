Data Attribute Use

HTML5 data-* attributes provide a convenient way to store custom data in HTML elements. The stored data can be accessed or modified using JavaScript. 

<div data-submitted="yes" class="user_profile">
  ... some content ...
</div>
  • The structure of a data attribute is data-*. The part after data- defines the key used to access it.
  • Any string-based data (including JSON) can be stored using data-* attributes.
