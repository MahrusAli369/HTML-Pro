Exercise 1 – Forms & Input Types
Create a registration form using HTML5 input types. Include:
        Text fields for First Name and Last Name
        Email input with validation
        Password and confirm password fields
        Phone number input
        Date of birth input
        Color picker for favorite color
        Range input for skill level (1–10)
        Submit button
Bonus: Make sure the form uses fieldset and legend to group related fields.

Exercise 2 – Image & Media
Create a webpage that demonstrates:
        Embedding 3 images, one local and two from the internet
        Use alt attributes properly
        One image should be responsive using srcset
        Add a video element with controls and fallback text
        Style the page with background color and padding

Exercise 3 – Semantic HTML
Build a mini blog page using semantic tags:
        <header> with the blog title and navigation links
        <main> containing at least two <article> elements
        Each <article> should have a <header> with a title and <time>
        Add a <section> inside one article for comments
        Include a <footer> for the page
        Extra: Apply basic CSS to differentiate header, main, and footer visually.

Exercise 4 – Forms Advanced Attributes
Create a form to submit a file and some metadata:
        Email input
        File input (accept images and pdf)
        Number input (quantity) with min, max, and step
        Week input
        Submit button
        Use target, enctype, and novalidate attributes correctly
Check: The form should demonstrate validation attributes (required, minlength, maxlength) and different input types.

Exercise 5 – Navigation & Layout
Design a webpage layout with:
        A navigation bar with 4 links, using <nav>
        Links displayed inline first, then in a list (<ul> & <li>)
        Multiple <section> blocks with headings and paragraphs
        A <footer> with links to privacy policy, terms, and contact
        Style the navigation links with hover effect and spacing
