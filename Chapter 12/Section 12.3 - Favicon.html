<link rel="shortcut icon" href="/favicon.ico" sizes="16x16 32x32" type="image/x-icon">

Use the MIME type image/png for PNG files and image/x-icon for icon (*.ico) files. A file named favicon.ico placed at the root of your website will usually be detected automatically, even without a <link> tag. However, if this file changes, browsers can sometimes be slow to update their cached version.
