Tabindex Values and Meaning

Value Meaning
negative The element is focusable, but cannot be reached through sequential keyboard navigation.
0 The element is focusable and reachable through keyboard sequential navigation; order is defined by platform convention.
positive The element is focusable and accessible via sequential keyboard navigation; order is determined by increasing tabindex values.

Try pressing Tab to navigate through the interactive elements above and observe how the focus changes.
