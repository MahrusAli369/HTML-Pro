Handling Disabled JavaScript

It is possible that the client browser does not support JavaScript or has it disabled for security reasons. To inform users that a script is supposed to execute on the page, the <noscript> tag can be used. The content inside <noscript> is displayed only when JavaScript is disabled. 

<script>
        document.write("Hello, world!");
</script>
<noscript>This browser does not support JavaScript.</noscript>
