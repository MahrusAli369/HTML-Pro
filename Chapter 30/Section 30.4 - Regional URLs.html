Using hreflang for Regional URLs

The hreflang attribute specifies the language (and optionally region) of a linked resource. It helps search engines and browsers identify language-targeted pages.

Try these regional versions of example sites:

Wikipedia (Urdu)
Wikipedia (Arabic)
Wikipedia (Spanish)
Wikipedia (English)
Wikipedia (French)
Wikipedia (German)

Reference: IETF RFC 5646 - Tags for Identifying Languages
