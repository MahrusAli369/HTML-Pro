Block with <blockquote> Element

The <blockquote> element is used for block-level quotes:

The answer is 42.

Using the cite Attribute

The cite attribute references the URL of the quoted source:

The answer is 42.

Note: Browsers typically don’t display the URL. If relevant, add a hyperlink:

The answer is 42.

Source: Hello World

Citation and Attribution

HTML5 allows the citation or attribution to be inside the blockquote using a <cite> or <footer> element:

The answer is 42.

Source: Hello World
