Older Browser Support for Data Attributes

Data attributes were introduced in HTML5 and are supported by all modern browsers. However, browsers developed before HTML5 do not recognize these attributes.

According to the HTML specification, any unrecognized attribute must be ignored by the browser during rendering. Developers leveraged this behavior to create non-standard attributes, which are attributes not defined in the HTML specification.

For example, the value attribute below is non-standard because it isn’t defined for the <img> tag: 

<img src="sample.jpg" value="test">

This means that although older browsers don’t officially support data attributes, they still preserve them. You can set and retrieve these values using the standard setAttribute() and getAttribute() JavaScript methods. However, the newer dataset property works only in modern browsers.
