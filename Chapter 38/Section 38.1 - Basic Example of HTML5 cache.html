Basic Example of HTML5 Cache

This example demonstrates using the HTML5 Application Cache to enable offline access to a webpage.

Step 1: Create your index.html file

<!DOCTYPE html>
<html manifest="index.appcache">

<body>
    <p>Content</p>
</body>

</html>

Step 2: Create the cache manifest file (index.appcache)

CACHE MANIFEST
index.html

Make sure both files are saved in the same folder. Once you open index.html in a browser, it will cache the page for offline use. Then, try going offline and reloading the page to see it served from the cache.

Note: HTML5 Application Cache is now deprecated in favor of Service Workers for modern offline web applications, but this example demonstrates the basic mechanism.
