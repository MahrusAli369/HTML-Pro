Keyboard Events

Keyboard events allow you to respond dynamically when a user presses, holds, or releases a key.

Event Attribute Description Interactive Demo
onkeydown Fires when a user is pressing a key
Waiting for keydown...
onkeypress Fires when a user presses a character key
Waiting for keypress...
onkeyup Fires when a user releases a key
Waiting for keyup...

Try typing in each input field. The feedback area below each field updates in real-time to show which key is pressed, held, or released.
