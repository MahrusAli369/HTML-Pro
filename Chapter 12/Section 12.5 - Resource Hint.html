Resource Hints – Live Demonstration

  1. Preconnect

    Establishes an early connection to a third-party server. In this demo, it preconnects to fonts.googleapis.com so Google Fonts load faster. 

    <link rel="preconnect" href="https://fonts.googleapis.com" />

  2. DNS Prefetch

    Resolves the DNS for fonts.gstatic.com before it's actually needed. 

    <link rel="dns-prefetch" href="https://fonts.gstatic.com" />

  3. Prefetch

    Downloads an image quietly in the background so it appears instantly when requested. 

    <link rel="prefetch" href="https://via.placeholder.com/400" />

  4. Prerender

    Loads and renders an entire page in the background (experimental). We’ll prerender https://example.com as a demo. 

    <link rel="prerender" href="https://example.com" />
    You can check this behavior in DevTools → Network → Look for example.com preloaded quietly.
