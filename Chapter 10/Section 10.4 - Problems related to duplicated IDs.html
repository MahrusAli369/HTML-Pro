Problems Related to Duplicated IDs

Having more than one element with the same ID is a hard-to-troubleshoot problem. The HTML parser will usually try to render the page anyway. Usually no error occurs, but the page could end up misbehaving. 

In this example:
<div id="aDiv">a div</div>
<div id="aDiv">b div</div>

CSS selectors still work:

#aDiv {
    color: red;
}

Output:

a
b

But JavaScript fails to handle both elements:

var html = document.getElementById("aDiv").innerHTML;

In this case, the "html" variable bears only the first div content ("a").
