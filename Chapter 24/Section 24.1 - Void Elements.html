HTML 4.01 / XHTML 1.0 Strict Void Elements

Tag Description
<area> Clickable area within an image map
<base> Specifies the base URL for relative links
<br> Inserts a line break
<col> Defines column properties in a table [deprecated]
<hr> Creates a horizontal line
<img> Embeds an image
<input> Defines an input field
<link> Links external resources like stylesheets
<meta> Provides metadata about the document
<param> Defines parameters for plugins

HTML5 Additions to Void Elements

Tag Description
<command> Represents a user command [obsolete]
<keygen> Generates encryption keys [deprecated]
<source> Defines media sources for audio/video elements
