application-name

Giving the name of the Web application that the page represents.

<meta name="application-name" content="OpenStreetMap" />

author

Set the author of the page:

<meta name="author" content="Mahrus Ali" />

Set the description of the page:

<meta name="description" content="Page Description" />

generator

<meta name="generator" content="HTML Generator 1.42" />

keywords

Set keywords for search engines (comma-separated):

<meta name="keywords" content="Keyword1, CSS, Java, JavaScript" />

Used by some search engines to determine the relevance of a page. Indexing often stops after the first ~20 words.
