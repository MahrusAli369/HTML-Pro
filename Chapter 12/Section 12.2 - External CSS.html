External CSS

The standard practice is to place CSS <link> tags inside the <head> tag at the top of your HTML document. This ensures your styles are loaded first, so the page displays properly as it loads instead of flashing unstyled content. The type attribute is not required in HTML5 because it defaults to text/css. 

<link rel="stylesheet" href="path/to.css" type="text/css">
and
<link rel="stylesheet" href="path/to.css">

Both examples above work the same way in HTML5. A less common method is to use an @import statement inside a <style> tag: 

<style type="text/css">
  @import url("path/to.css");
</style>

<style>
  @import url("path/to.css");
</style>
