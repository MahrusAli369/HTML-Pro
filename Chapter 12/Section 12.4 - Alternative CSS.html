Alternative CSS

<link rel="alternate stylesheet" href="path/to/style.css" title="yourTitle">

Some browsers allow alternate style sheets to be offered. By default, they will not be applied automatically, but users can activate them through browser settings.

Firefox lets users select the stylesheet via View > Page Style. Internet Explorer also supports this (from IE 8 onward) through the same menu. Chrome, however, requires an extension to enable this feature (as of version 48). A webpage can also provide its own interface to let users switch styles dynamically.
