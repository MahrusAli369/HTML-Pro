Default (native order)

Positive tabindex values insert the element at the specified position in the tabbing order. Elements without a preference (i.e., tabindex="0" or native elements like button and a) follow after those with positive values.

Important Considerations

Using positive tabindex values is generally discouraged because it can disrupt expected keyboard navigation and confuse users relying on screen readers. A better approach is to arrange elements naturally in the DOM to reflect the desired navigation order.
