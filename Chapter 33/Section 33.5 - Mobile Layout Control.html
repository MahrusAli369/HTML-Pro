Section 33.5 - Mobile Layout Control

<meta name="viewport" content="width=device-width, initial-scale=1">

Attribute Description
width          The width of the virtual viewport of the device.
               Values1: device-width or a fixed pixel width (e.g., 480)

height         The height of the virtual viewport of the device.
               Values2: device-height or a fixed pixel height (e.g., 600)

initial-scale  Sets the initial zoom when the page loads (1.0 = no zoom)

minimum-scale  Minimum zoom level allowed (1.0 = no zoom)

maximum-scale  Maximum zoom level allowed (1.0 = no zoom)

user-scalable  Allows zooming: yes or no (default is yes)

Notes:

1 — The width property can be set in pixels (width=600) or by using device-width which represents the actual physical width of the screen.

2 — The height property follows the same rule: device-height or an exact pixel height.
