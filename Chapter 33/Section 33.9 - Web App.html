Section 33.9 - Web App

You can configure your website or web app so users can add it to their deviceâ€™s homescreen as a shortcut,
making it launch in a clean, full-screen "app mode," especially in Chrome for Android.

Below meta tags allow full-screen display:

Android Chrome:
    <meta name="mobile-web-app-capable" content="yes">

iOS:
    <meta name="apple-mobile-web-app-capable" content="yes">

You can also define the status bar and browser UI color:

Android Chrome:
    <meta name="theme-color" content="black">

iOS:
    <meta name="apple-mobile-web-app-status-bar-style" content="black">
