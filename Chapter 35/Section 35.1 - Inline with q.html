Inline with <q> Element

The <q> element can be used for a quote that is part of a sentence:

She wrote The answer is 42. and everyone agreed.

Quotation Marks Behavior

HTML 4.01: Quotation marks should not be added; user agents render them automatically.

HTML5: Quotation marks must not be added; user agents render them automatically.

Using the cite Attribute

The cite attribute can reference the URL of the quoted source:

She wrote The answer is 42. and everyone agreed.

Note: Browsers usually don’t display the URL. If relevant, add a hyperlink:

She wrote The answer is 42 and everyone agreed.

More Examples

John said We need to rethink our strategy. before the meeting started.

As the report stated, Efficiency improved by 20% last quarter.
